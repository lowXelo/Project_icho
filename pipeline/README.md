Folder for pipeline 

simple_pipeline : registration by simple general method like translation or rotation

ILK_pipeline_v1 : pipeline that take a .fits and look for the best ILK params to register, then shows graph to see CC evolution
ILK_MI_pipeline : Same that v1 but with mutual information (MI) rather than corr. coef. 

ILK_pipeline_V2 : introduce three functions to do what v1 does, and then allow to register images more generally.
